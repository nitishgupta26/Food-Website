## Features:
- User can view menu category wise.
- Users can send his/her order information from Contact form.
- Mobile responsive
- Deployed on heroku. 


## Technologies used:

**FrontEnd:**     HTML, CSS, Javascript.

**BackEnd:** Node.js, Express.js, Mongodb, Pug.js.

  
## Installation:

Install my-project with npm

```bash
  npm Install
  npm test
  npm run build
  npm run eject
```
